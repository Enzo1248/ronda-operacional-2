# Ronda Operacional

Projeto React simples para realizar rondas de bancadas e gerar slides (.pptx).

## Como rodar
1. Instale o Node.js (versão LTS).
2. No terminal, dentro desta pasta:
   ```bash
   npm install
   npm start
   ```
3. Abra `http://localhost:3000` no navegador.

O botão **Gerar PPTX** cria um arquivo `.pptx` com a logo no canto superior direito e imagens anexadas quando existirem.
